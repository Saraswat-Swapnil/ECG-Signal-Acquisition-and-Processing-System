# ECG Signal Acquisition and Processing Circuit

## Project Overview

This project presents the design, simulation, and hardware implementation of a low-cost ECG (Electrocardiogram) Signal Acquisition and Processing Circuit.

The system acquires weak bioelectrical signals generated by the heart and processes them through multiple analog signal conditioning stages, including amplification, high-pass filtering, low-pass filtering, and 50 Hz notch filtering. The objective is to obtain a clean ECG waveform suitable for monitoring, analysis, and future digitization using ADC-based embedded systems.

The project combines biomedical instrumentation principles with analog electronics and signal processing techniques.

---

## Objectives

- Acquire low-amplitude ECG signals from the human body.
- Amplify weak biopotential signals.
- Remove baseline wander and DC offsets.
- Eliminate 50 Hz power-line interference.
- Suppress high-frequency noise and artifacts.
- Validate circuit performance through simulation and hardware testing.
- Develop a low-cost ECG front-end suitable for further digital processing.

---

## System Architecture

The ECG acquisition chain consists of the following stages:

### 1. Signal Acquisition

- ECG electrodes collect cardiac electrical activity.
- Differential voltage signals are extracted from the body.

### 2. Instrumentation Amplifier

- High common-mode rejection ratio (CMRR)
- Amplification of weak ECG signals
- Improved noise immunity

### 3. High-Pass Filter (HPF)

Purpose:
- Remove baseline drift
- Eliminate DC offset
- Reduce respiration-induced variations

Hardware cutoff frequency:

```text
≈ 0.5 Hz
```

### 4. Notch Filter

Purpose:
- Suppress 50 Hz power-line interference

Target notch frequency:

```text
50 Hz
```

### 5. Low-Pass Filter (LPF)

Purpose:
- Remove high-frequency noise
- Reduce muscle artifacts (EMG)
- Preserve ECG frequency components

Hardware cutoff frequency:

```text
≈ 150 Hz
```

### 6. Final Output Stage

- Cleaned ECG waveform
- Suitable for ADC interfacing
- Ready for digital signal processing
